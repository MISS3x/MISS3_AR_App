# AI Codebase & External References

This document serves as a repository of core technical documentation, architecture patterns, and external API references for the MISS3 AR App.

## Android ARCore Documentation
When dealing with Android-specific AR implementation, tracking, planes, or lighting features, refer to the official Google ARCore documentation:
* **ARCore Developer Guide:** [https://developers.google.com/ar/develop](https://developers.google.com/ar/develop)
* Contains specific functions, methodologies, and Android examples for spatial computing.

## iOS ARKit Documentation
When dealing with iOS-specific AR implementations, refer to the official Apple ARKit libraries:
* **ARKit Developer Guide:** [https://developer.apple.com/augmented-reality/arkit/](https://developer.apple.com/augmented-reality/arkit/)
* Used internally by ViroReact via the SceneKit bridge.
